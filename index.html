Demo Project

System Entry Interface

This system provides two primary functions:

  • Report evaluation
  • Report generation

Report Evaluation

Includes Login Page and Survey Page.

New functionality:

  • Supports 3D medical imaging
  • Various visualization operations for medical images

Image Display and Interactions

Interactive features include:

  • Pan
  • Zoom
  • Rotate
  • Adjust contrast
  • View DICOM Series

Report Generation Page

The interface includes:

  • Image Upload and Preview Area: Supports various formats like .dicom, .png, .jpg, and .nii.gz, including batch uploads from folders.
  • Report Generation Area: Displays previews and supports pan, zoom, contrast, rotation, and fullscreen interaction.
